# 整体视觉方向系统

视觉方向必须服务已确认的故事线和企业品牌，不从“科技蓝、发光线条、大屏幕”等通用套路开始。

## 转译步骤

1. 从品牌手册提取已有色彩、字体、图形和禁用规则。
2. 从故事线提取核心动词、空间意象和情绪变化。
3. 为每个叙事阶段确定主色、材质、光环境、图形语言和媒体强度。
4. 用同一套视觉语法连接各展区，同时保留重点展区的节奏变化。
5. 检查视觉建议是否能被空间、预算和维护条件支持。

## 输出维度

- 色彩：品牌主色、辅助色、强调色及使用比例原则。
- 材质：基底、重点、透光、反射和触感材料的概念建议。
- 灯光：基础照明、重点照明、氛围光和媒体亮度关系。
- 图形：品牌图形、数据图、时间线、地图和案例展示语言。
- 多媒体：使用目的、内容长度、交互方式和无设备替代方案。
- 无障碍：文字可读性、对比度、字幕、通行和交互高度提醒。

## 文字与品牌资产

AI 效果图只表现空间、结构、材质和灯光。企业 LOGO、中文标题、数据、二维码和荣誉名称默认在后期使用矢量设计工具添加，不要求图像模型准确绘制。

## 反通用化检查

删除无法从品牌事实、故事线或空间条件解释的装饰。若视觉方向替换公司名称后仍适用于任何企业，继续提炼企业专属意象和内容结构。
