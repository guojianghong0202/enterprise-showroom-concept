# 效果图提示词交接

## 可选调用

先检查当前环境是否存在 `advertising-render-prompts`。存在时，把每个展区的交接卡交给该 Skill 深化，不复制其完整工作流。不存在时，使用本文件的通用模板；不要自动安装依赖。

## 标准交接卡

```text
展区编号与名称：
图像用途：概念效果参考 / 提案视觉 / 风格探索
目标模型：已知模型或“通用”
空间条件：面积、层高、入口、结构、视线和不可改造条件
故事线角色：该展区在整体叙事中的作用
核心内容：事实摘要及证据编号
访客与情绪目标：
空间构成：墙面、展台、展柜、屏幕、装置和通道关系
色彩与材质：
灯光：
相机与画幅：
文字政策：中文、LOGO、数据、二维码后期添加
避免项：
待确认项：
```

## 通用提示词骨架

```text
[企业品牌展厅的具体展区]，位于[空间与结构条件]，承担[故事线角色]，
以[空间构成]呈现[核心内容]，使用[色彩与材质]，采用[灯光策略]，
[相机位置、焦段感和画幅]，专业展陈设计，真实尺度，合理通道，
可实施感的结构与收口，概念效果参考；画面内不生成准确中文、LOGO、二维码或企业数据。
```

## 负面约束

根据场景选择：错误透视、尺度失真、堵塞通道、悬浮结构、不可安装构件、过度镜面、无意义大屏、乱码中文、变形 LOGO、虚构企业数据、舞台化灯光、廉价科技感、同质化蓝色光带。

## 交付规则

每个展区分别输出正向提示词、避免项、后期文字与 LOGO 说明、关键假设和待确认项。没有实际调用图像工具时，不声称已经生成图片。
